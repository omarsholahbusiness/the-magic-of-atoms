# إشعارات الكورسات الجديدة (New course release notifications)

الطلاب يمكنهم تفعيل إشعارات المتصفح (Google Chrome وغيره) ليصلكم تنبيه عند نشر كورس جديد يناسب صفكم.

## تفعيل الإشعارات من الإدارة

1. **إنشاء مفاتيح VAPID** (مرة واحدة):

```bash
npx web-push generate-vapid-keys
```

ستظهر مفتاحان: **Public Key** و **Private Key**.

2. **إضافتهم في `.env`**:

```env
# Web Push (إشعارات الكورسات الجديدة)
NEXT_PUBLIC_VAPID_PUBLIC_KEY="ضع_المفتاح_العام_هنا"
VAPID_PRIVATE_KEY="ضع_المفتاح_الخاص_هنا"
```

3. **إعادة تشغيل السيرفر** بعد تعديل `.env`.

## السلوك

- في **لوحة تحكم الطالب** يظهر زر **"تفعيل إشعارات الكورسات الجديدة"** إذا لم يكن الإشعار مفعّلاً.
- عند الضغط يطلب المتصفح الإذن ثم يحفظ الاشتراك.
- عند **نشر كورس** (من المعلم)، يُرسل إشعار للمتصفح لكل الطلاب الذين صفهم مطابق لصفوف الكورس (أو كل الطلاب إذا الكورس بدون صفوف "الكل").

## ملاحظات

- الإشعارات تعمل في المتصفحات الداعمة (Chrome, Firefox, Edge).
- يجب أن يكون الموقع يعمل عبر **HTTPS** في الإنتاج (أو localhost للتجربة).
